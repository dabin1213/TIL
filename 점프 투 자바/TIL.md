```
return = 메서드에서 값을 반환하는 작업

equals = 문자열 2개가 같은지를 비교하여 결괏값을 리턴
(문자열의 값을 비교할 때는 반드시 equals를 사용)

indexOf = 문자열 시작되는 위치 출력

contains = 특정 문자열 포함되어 있는지 여부 
```
```

 시작 위치에서 끝 위치까지의 문자를 출력)
```
